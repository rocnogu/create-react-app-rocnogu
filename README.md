# What it contains

ReactJS basic App

## To run the app

Go Into the project directory and run

#### `npm i `

to install dependencies

* and

#### `npm start`

to develop locally

Opens [http://localhost:3000](http://localhost:3000) in the browser.

### To run React app on new port

#### `export PORT=8000`
 in the console

* or

####  ` "start": "set PORT=8000 && react-scripts start" `  
in package.json
